This Github includes files used in 
https://ak321blog.wordpress.com/2016/02/02/kaggle-airbnb-competition/

The "RandomSearch" file just applies random parameter search for the XGBoost classifier.
This file is modified from a code shared on the kaggle forum (see the post in my blog, link above).

The "Session" file is to do a simple statistics on the session fata.
This data can be the extension of the train/test data.
One post on the forum claims this improves the performance (I haven't tried)
